# ilovejuejin-tools


欢迎大家推荐【开发】时的常用工具，fork此仓库并提PR


### 注意事项

1. 必须确保你推荐的常用工具的url可以被【iframe】，否则无法通过。
2. 请确保提交的工具和日常开发有关。
2. 更改tools.json时注意格式

~~~javascript
"title":"工具的名称",
"url":"工具的url，请确保可以被iframe引入",
"description":"详细说明你推荐此工具的原因，不超过44个汉字",
"recommendby":"你的掘金账号id，个人主页url后面那串数字"
~~~
